# PackDisabler

Hypixel Skyblock forces a custom resource pack to every connecting player. PackDisabler intercepts it at the network level, silently tells the server it loaded, and keeps your client on vanilla visuals.


## Why use PackDisabler?

- The Skyblock pack slows down loading time (more DT YEY!)
- The Official Hypixel Skyblock pack is dogshit like omg
